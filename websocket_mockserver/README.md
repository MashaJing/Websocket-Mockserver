# Websocket mockserver (server)

## Сборка образа

```bash
docker build -t mock-server .
```

## Параметры запуска

При запуске сервера можно указать порт несколькими способами:

1. Через аргументы командной строки

```bash
python main.py -p 8080
```

или

```bash
python main.py --port 8080
```

2. Через переменную окружения `PORT` (если аргумент командной строки не передан):

```bash
docker run -d -p 9000:9000 -e PORT=9000 ...
```

или

```bash
export PORT=8080
python main.py
```

3. Если ни аргумент командной строки, ни переменная окружения не заданы, сервер запустится на порту 8000.
